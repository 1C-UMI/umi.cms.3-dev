Buttons [Кнопки].

Для объявления кнопки достаточно задать класс .button.

Кнопки имеют классы-модификаторы изменяющие:

Размер. Классы tiny, small, large.
  • Ширину. Классы expand, wide, square.

  • Цвет. Классы secondary, success, alert, white, flat.

  • Радиус. Классы round, radius.

  • Состояние. Классы disabled, loading.

Buttons Group [Группа кнопок].

Для того чтобы определить группу кнопок используется класс button-group

Объединить группы кнопок можно обернув их в элемент с классом button-bar
